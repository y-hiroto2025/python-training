# Python Training Log
このリポジトリは、Geminiで生成したPythonの練習問題を使ってプログラミングの勉強をしたり、自分で思いついた便利だと思ったコードを書いた記録です。

## 1. 簡単なRPGバトルシミュレート
* **内容:** プレイヤーとモンスターが戦うターン制のバトル
* **学んだこと:** クラス、ループ処理、条件分岐

## 2. ファイル操作を使った簡単な記憶システム
* **内容:** ユーザーが入力した言葉と意味をテキストファイルに保存する
* **今後の予定:** txtファイル形式からJSON形式への変更

## 3. numpyを使った数値分析
* **内容:** 正規分布に従うランダムな数値1000個の統計データを算出（最大・最小・平均など）
* **学んだこと:** Numpy

## 4. データの見える化
* **内容:** NumPyで作ったデータをグラフとして表示する
* **学んだこと:** NumPy, Matplotlib

## 5. 汚れたログをきれいにする
* **内容:** 汚れたログデータから不要な記号を取り除き、必要な情報だけを抽出する

## 6. 自動で複数のウェブページとソフトを起動する
* **内容:** よく使うWebサイト（AtCoder, GitHub等）とVS Codeを一括起動する
* **学んだこと:** webbrowser, subprocess
